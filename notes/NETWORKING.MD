## Socket Programming in Python :

Sockets and the socket API are used to send messages across a network. They provide a form of inter-process communication (IPC). The network can be a logical, local network to the computer, or one thatâ€™s physically connected to an external network, with its own connections to other networks. The obvious example is the Internet, which we connect to via your ISP.

```python

import socket
s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

```
Here we made a socket instance and passed it two parameters. The first parameter is AF_INET and the second one is SOCK_STREAM. AF_INET refers to the address-family ipv4. The SOCK_STREAM means connection-oriented TCP protocol. The primary socket API functions and methods in this module are:

```

socket()
.bind()
.listen()
.accept()
.connect()
.connect_ex()
.send()
.recv()
.close()

```

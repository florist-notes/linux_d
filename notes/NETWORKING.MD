<th>ip route show</th>
<td>Use the IP route to print or display the routing table.<br /></td>

<tr>

<td>By default, ping runs in an infinite loop. To send a defined number of packets, use -c flag<br /></td>

<tr>

</tr>

<th>iftop</th>
